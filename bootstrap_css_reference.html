Integration & Setup

Start utilizing HTML5, CSS3 variables, and Bootstrap 5.3 CDN configurations directly in your web files.

Method Source Integration Tag Use Case
Bootstrap CSS CDN <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet"> Add in <head> for layout and pre-made styles.
Bootstrap JS CDN <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script> Add before </body> for modals, dropdowns, tooltips.
Custom CSS Link <link rel="stylesheet" href="style.css"> Add in <head> *after* Bootstrap for custom override styles.
Standard Starter Template (HTML) 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Bootstrap Starter</title>
  <!-- Bootstrap CSS -->
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
  <!-- Custom Styles -->
  <link rel="stylesheet" href="custom.css">
</head>
<body>
  <h1 class="text-primary text-center my-5">Hello World!</h1>

  <!-- Bootstrap Bundle JS with Popper -->
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>

HTML Core Attributes

Global attributes that modify the behavior or style representation of standard HTML tags.

HTML Attribute Purpose CSS Equivalent Selector Code Example
class Assigns class names for styling or grouping elements. Can take multiple classes. .classname { } <div class="text-center p-3">
id Unique identifier for an element. Essential for CSS specificity, anchor targets, and JavaScript bindings. #idname { } <section id="contact">
style Applies inline CSS directly to the element. Has the highest specificity weight. Direct rendering overrides stylesheet <div style="color: red; padding: 10px;">
title Tooltip popover displayed when hovering over an element. Good for extra info. Browser default rendering <button title="Click to Submit">
hidden Hides the element from the display. Equivalent to display: none. [hidden] { display: none; } <input type="text" hidden>
data-* Custom user attributes allowing metadata to be stored on elements. Critical for Bootstrap scripts. [data-*] (Attribute Selector) <div data-user-role="admin">

CSS & The Box Model

Understanding how the browser interprets size and layouts using standard CSS box properties.

Content Box
Concept CSS Property Bootstrap Mapping Description
Margin margin m-*, mt-*, mb-* Spacing outside the border line. Pushes adjacent elements away.
Border border border, border-primary Line wrapping around padding and content box. Can define width, style, color.
Padding padding p-*, px-*, py-* Spacing inside the element between the content and the border line.
Content width / height w-100, h-50 The actual size of text, images, or child nested objects inside.

Data Attribute Key Values Target Element Purpose
data-bs-toggle "modal", "collapse", "dropdown", "tooltip", "carousel" Specifies which JavaScript plugin is hooked to the element.
data-bs-target CSS Selector ID (e.g., "#exampleModal") Points to the unique ID of the component element that should change.
data-bs-dismiss "modal", "alert", "offcanvas" Instructs the plugin to close/remove the active parent container.
data-bs-slide / data-bs-slide-to "prev", "next" / Index numbers (e.g., "0") Controls direction movement for Carousels.
data-bs-ride "carousel" Auto-starts slide transitions upon page load initialization.

ARIA & Accessible Attributes

Integrate essential attributes to guarantee proper semantic access for assistive technologies (screen readers).

ARIA Attribute Type / Value Semantic Context Mapped Screen Reader Action
role "dialog", "alert", "navigation", "tablist" Assigns semantic meaning to generic layout wrappers. Declares structure type explicitly.
aria-expanded "true", "false" Identifies whether a submenu or accordion section is visible. Announces expanded or collapsed status.
aria-labelledby Target Element ID reference Links container label descriptions directly to internal header text tags. Reads associated element's content as label description.
aria-hidden "true", "false" Excludes helper items (icons/backdrop divs) from readers. Completely ignores elements from text audio tracks.
tabindex "0", "-1", "1" Controls focus behavior during keyboard tab cycling. Binds key navigation.

Custom CSS Variable Integration

Override Bootstrap's design colors and border ratios using root custom CSS variables (no Sass compiling required).

CSS Variables Overriding & Utility Classes (CSS) 
/* custom.css */
:root {
  /* Override Bootstrap Brand Colors using native CSS variables */
  --bs-primary: #6366f1; /* Neon Purple */
  --bs-primary-rgb: 99, 102, 241;
  
  --bs-success: #10b981; /* Emerald Green */
  --bs-success-rgb: 16, 185, 129;

  /* Custom component override */
  --bs-border-radius: 12px;
}

/* Specificity override card targeting without utilizing !important */
.custom-card {
  border-radius: var(--bs-border-radius);
  background: linear-gradient(135deg, rgba(var(--bs-primary-rgb), 0.1), rgba(var(--bs-success-rgb), 0.05));
  border: 1px solid rgba(var(--bs-primary-rgb), 0.15);
  transition: transform 0.2s ease;
}

.custom-card:hover {
  transform: translateY(-4px);
  border-color: var(--bs-primary);
}
